Information

Limit & Request CPU:
How much CPU will the total combined resources in your namespace require? This is measured in CPU cores, so you will input "1" for 1 core, "2" for 2 cores, and so on.
request.cpu   =   (numPods * minCoresPerPod)
limit.cpu         =   (numPods * maxCoresPerPod)

Limit & Request Memory:
How much memory, or RAM, will the total combined resources in your namespace require? This is measured in either Megabytes or Gigabytes.
request.memory   =   (numPods * minRAMPerPod)
limit.memory         =  (numPods * maxRAMPerPod)

Limit & Request Storage:
How much local storage, if any, will the total combined resources in your namespace require? The input requirements are the exact same as those associated with Memory i.e. 10Gi for 10 Gigabytes, etc.
request.storage  =   (numPods * minLocalStoragePerPod)
limit.storage       =   (numPods * maxLocalStoragePerPod)
